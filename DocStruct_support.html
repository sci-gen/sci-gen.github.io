This page provides support information for DocStruct.

DocStruct is a desktop application for opening PDF files, previewing pages, splitting spreads, and exporting processed PDF files.

We want DocStruct to feel dependable for researchers, archive work, and scanned-document cleanup. If you run into a problem, we want to receive enough detail to investigate it properly.

Support Scope

Support is currently lightweight, but we do review incoming reports and use them to improve stability, usability, and release quality.

The most useful messages are:

  • bug reports and reproducible failures
  • workflow pain points or usability issues
  • feature requests related to scanned PDF cleanup
  • questions about privacy policy or support policy

Response times are not guaranteed, and individual setup assistance may be limited, but clear reports help us prioritize fixes and polish future releases.

Information to include in your message

To help us investigate faster, please include the following when relevant:

  • app name: DocStruct
  • app version
  • Windows version
  • what happened
  • steps to reproduce the issue
  • error message or unexpected behavior
  • whether the issue happened on a large scanned PDF or a smaller document

What you can expect

  • privacy and policy questions can usually be answered directly
  • clear bug reports can be used to guide fixes in upcoming updates
  • feature requests are reviewed, especially when they improve scanned-document workflows
